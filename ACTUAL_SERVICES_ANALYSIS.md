## **CORRECTED MED SPA UNDERSTANDING**

**NOT "wellness/relaxation spa"** - These are **medical aesthetic treatments**:

- **Gold Facial** - Anti-aging, collagen production, medical-grade
- **HydraFacial** - Medical-grade skin resurfacing
- **LED Light Therapy** - Medical device for acne/anti-aging
- **Dermaplaning** - Medical-grade exfoliation
- **Threading/Waxing** - Precision hair removal services

**Target**: Clients seeking **medical-grade aesthetic improvements**, not relaxation.

## **CORRECTED MAKEUP STUDIO UNDERSTANDING**

**NOT "cosmetology school"** - Professional makeup services + limited education:

- **Bridal/Event makeup** - Professional application for occasions
- **Lash extensions** - Semi-permanent beauty enhancement
- **Eyebrow styling** - Professional shaping/tinting

## **REAL HERO SECTION DISAMBIGUATION NEEDS**

### **Current Problem:**
Homepage shows domain-based navigation but users think in **outcome terms**:

- ❌ "I need hair services" 
- ✅ "I have a wedding in 2 months"

- ❌ "I need makeup services"
- ✅ "I want to look professional for work"

- ❌ "I need med spa services"  
- ✅ "I want better skin"

### **Better Disambiguation Strategy:**

#### **Outcome-Based Entry Points:**
```
"What would make you feel amazing?"

├── I have a special event coming up
│   → Bridal packages, event makeup, pre-event facials
│   → Cross-domain packages (hair + makeup + skin prep)
│
├── I want to look more polished daily  
│   → Lash extensions, eyebrow styling, natural makeup
│   → Regular maintenance services
│
├── I need regular maintenance
│   → Cuts, color touch-ups, threading, lash fills
│   → Subscription/recurring services
│
├── I want to improve my appearance
│   → Transformations, anti-aging treatments, dramatic changes
│   → Multi-service packages
│
└── I want better skin
    → Medical-grade facials, LED therapy, dermaplaning
    → Skin improvement programs
```

## **CONSULTATION FLOW STRATEGY**

### **Goal**: Understand **user motivation** → Guide to **appropriate services** → **Facilitate booking**

### **Real Questions to Ask:**
1. **"What's bringing you in?"** (Event, maintenance, improvement, etc.)
2. **"When do you need this?"** (Timeline affects service selection)
3. **"What's your experience level?"** (Affects service complexity)
4. **"What's your budget range?"** (Affects package recommendations)

### **Service Recommendation Logic:**
- **Event + Timeline** → Event packages with prep services
- **Maintenance + Frequency** → Regular service subscriptions  
- **Improvement + Budget** → Transformation packages
- **Daily enhancement** → Beauty enhancement services
