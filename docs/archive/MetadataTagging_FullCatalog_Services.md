## Optimal Metadata Tag System

### Experience Level Tags
**Purpose:** Filter by user comfort and skill level

- **`beginner-friendly`** (22 services): All basic cuts, blow outs, basic threading, basic waxing, dinner party makeup, basic facials
- **`intermediate-level`** (18 services): Styling services, partial highlights, toner, advanced threading/waxing, formal makeup, anti-aging facials
- **`advanced-expertise`** (11 services): Full highlights, color services, perm, keratin, lash extensions, bridal makeup, premium facials

### Time Investment Tags
**Purpose:** Match services to available appointment time

- **`quick-service`** (15 services): Basic threading (eyebrow, upper lip, chin), basic waxing (underarms, bikini), eyebrow/lash tinting, men's cuts, kids cuts
- **`standard-duration`** (20 services): Women's haircuts, blow outs, basic styling, partial highlights, basic facials, dinner party makeup
- **`extended-session`** (16 services): Full highlights, color services, perm, keratin, premium facials, formal/bridal makeup, full body waxing

## Rule-Based Filtering Implementation

### User Profile → Service Filtering

```typescript
interface UserProfile {
  experienceLevel: 'beginner' | 'intermediate' | 'advanced'
  timeAvailable: 'quick' | 'standard' | 'extended'
  occasionContext: 'everyday' | 'professional' | 'special-occasion' | 'event'
  budgetRange: 'budget-friendly' | 'moderate-price' | 'premium-service'
  maintenancePreference: 'low' | 'moderate' | 'high'
  enhancementGoals: string[] // ['facial-enhancement', 'hair-transformation', etc.]
}

function filterServices(userProfile: UserProfile, allServices: Service[]) {
  return allServices.filter(service => {
    // Experience Level Filter
    if (userProfile.experienceLevel === 'beginner' && 
        !service.tags.includes('beginner-friendly')) {
      return false
    }
    
    // Time Constraint Filter
    if (userProfile.timeAvailable === 'quick' && 
        service.tags.includes('extended-session')) {
      return false
    }
    
    // Budget Filter
    if (userProfile.budgetRange === 'budget-friendly' && 
        service.tags.includes('premium-service')) {
      return false
    }
    
    // Enhancement Goals Filter
    const hasMatchingGoal = userProfile.enhancementGoals.some(goal =>
      service.tags.includes(goal)
    )
    if (userProfile.enhancementGoals.length > 0 && !hasMatchingGoal) {
      return false
    }
    
    return true
  })
}
```

      a.tags.includes(goal)).length
    const bGoalBonus = userProfile.enhancementGoals.filter(goal => 
      b.tags.includes(goal)).length
