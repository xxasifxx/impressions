# Urgency Level Tags

## 💡 **Usage Guidelines**

### **For Appointment Scheduling:**
- **IMMEDIATE/HIGH urgency** → Priority booking, same-week availability
- **MEDIUM urgency** → Standard booking, 2-4 week window
- **LOW/FLEXIBLE urgency** → Fill-in appointments, flexible scheduling

### **For Client Communication:**
- Use urgency tags to set realistic expectations
- Help clients understand optimal timing
- Prioritize urgent needs in consultations
- Plan ahead for predictable urgent needs

### **For Business Operations:**
- Reserve emergency slots for immediate urgency
- Plan seasonal capacity for predictable urgent needs
- Use flexible services to fill schedule gaps
- Train staff on urgency assessment

### **For Service Recommendations:**
- Match urgency level to client timeline
- Suggest alternatives for high-urgency situations
- Plan maintenance to prevent urgent situations
- Educate clients on optimal timing

**These urgency tags help prioritize services and manage client expectations while maintaining simple tag-based organization.**
